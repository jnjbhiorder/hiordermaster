1. 색상 테마 비교

Red, Black, Gray 세 가지 테마의 특징과 장단점을 비교하여 브랜드 이미지에 맞는 최적의 색상을 선택합니다.

Red 테마

식욕을 자극하는 강렬한 이미지

Black 테마

고급스럽고 세련된 이미지

Gray 테마

중립적이고 안정적인 이미지

2. 상품 진열 수에 따른 UI 변화

한 번에 보여주는 상품의 개수(1개, 2개, 3개)에 따라 사용자 인터페이스가 어떻게 달라지는지 비교합니다.

1개 진열

상품 설명을 함께 노출하여 상세 정보 전달에 용이합니다.

2개 진열

균형감 있는 배치로 시각적 안정감을 줍니다.

3개 진열

다양한 상품을 한눈에 비교하기 좋습니다.

3. 인사말 사용 여부

초기 화면에 환영 메시지를 표시할 때와 표시하지 않을 때의 사용자 경험 차이를 분석합니다.

인사말 사용

"환영합니다" 메시지와 안내 문구를 노출하여 사용자에게 친근한 첫인상을 줍니다.

인사말 미사용

상품 목록만 바로 노출하여 사용자가 빠르게 메뉴를 탐색할 수 있습니다.

4. 주문 번호 노출 여부

주문 완료 후 주문 번호를 표시하는 경우와 표시하지 않는 경우의 차이점을 비교합니다.

주문 번호 출력

주문 번호(예: 209)를 명확히 표시하여 고객과 직원 간의 소통을 돕습니다.

주문 번호 미출력

주문 번호 없이 주문 내역만 표시하여 화면을 간결하게 유지합니다.

5. 대분류 사용 여부

메뉴 카테고리에 상위 분류(대분류)를 추가했을 때와 그렇지 않을 때의 UX를 비교합니다.

대분류 사용

'식당', '카페' 등 상위 분류를 추가하여 메뉴 구조를 더 체계적으로 만듭니다.

대분류 미사용

기본 상품 분류('고기류', '식사류' 등)만 사용하여 메뉴 접근 단계를 줄입니다.

6. 상품 노출 방식 비교

모든 상품을 한 번에 보여주는 방식과 카테고리별로 나누어 보여주는 방식의 차이점을 분석합니다.

상품 모두 노출

스크롤 시 모든 상품 분류가 순서대로 노출되어 전체 메뉴를 쉽게 훑어볼 수 있습니다.

상품 분류별 노출

선택한 상품 분류의 메뉴만 고정되어 보여주므로 특정 카테고리에 집중하기 좋습니다.
